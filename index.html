Las Pruebas de Software.

L

as pruebas de software son el proceso de evaluación de un sistema o sus componentes con la intención de verificar si cumple los requisitos especificados o no. Las pruebas se ejecutan en un sistema con el fin de identificar lagunas, errores, o los requisitos que faltan segun las necesidades reales.

La informacion suministrada en este sitio le dará un entendimiento básico de las pruebas de software, sus tipos, métodos, niveles y otras terminologías relacionadas.

Las pruebas de software se integran dentro de las diferentes fases del ciclo del software dentro de la Ingeniería de software. Así se ejecuta un programa y mediante técnicas experimentales se trata de descubrir que errores tiene. Para determinar el nivel de calidad se deben efectuar unas medidas o pruebas que permitan comprobar el grado de cumplimiento respecto de las especificaciones iniciales del sistema.

Objetivo General del OVI

Crear un sitio web para estudiar y comprender las pruebas de software, así como sus principios generales, como parte integral de garantizar la calidad del producto, validando que cumpla las especificaciones para las que fue diseñado.

Objetivos especificos de este OVI

  1. Conocer las diferentes técnicas de pruebas de software.
  2. Dar a conocer la importancia de las pruebas de software.
  3. Identificar los conceptos generales que están presente en un entorno de pruebas de software.
